# 赚钱离场

VS Code / Cursor 股票行情看盘插件。支持 A 股 / 港股 / 美股实时行情，以及环球指数、期货、贵金属；自选股分组管理、拼音首字母搜索订阅、拖拽排序、备注与涨跌幅排序，悬停显示 MA 均线，左键打开详情页（分时 / 日 K / 周 K），可选状态栏行情与价格提醒。

灵感来自 LeekFund（韭菜盒子），专注**精简自选股**，不含基金、持仓盈亏、资讯等重型功能。

## 功能

- **自选股面板** — 侧边栏 TreeView，实时显示现价、涨跌幅、成交额；支持分组嵌套、拖拽排序、备注；悬停可查看昨收、今开、最高/最低相对昨收涨跌幅，以及 MA5 / MA10 / MA20
- **股票详情页** — 左键点击自选股打开编辑器内详情；A 股 / 港股支持分时图（含 9:15 起盘前竞价），A 股 / 港股 / 美股支持日 K、周 K 及 MA 均线；可跳转浏览器查看完整行情页
- **价格提醒** — 为单只股票设置上限 / 下限，穿越时弹出通知；已设提醒的股票行末显示铃铛图标
- **搜索订阅** — 输入名称、代码或拼音首字母缩写实时搜索，一键添加 A 股 / 港股 / 美股，以及环球指数、期货、贵金属
- **涨跌幅排序** — 按涨幅降序 / 跌幅升序排列，可随时取消恢复手动顺序
- **状态栏行情** — 底部状态栏实时显示自选指数或股票价格，支持红绿涨跌色或黑白模式
- **市场概览** — 状态栏显示 A 股涨跌家数、平盘家数与两市成交额，默认开启，可单独关闭
- **配置导入导出** — 标题栏一键导出 / 导入自选股与分组配置

## 安装

### 从扩展市场安装（推荐）

在 VS Code / Cursor 中打开扩展面板，搜索 **赚钱离场** 并安装。

### 从 VSIX 本地安装

1. 获取 `take-home-x.x.x.vsix` 安装包
2. 扩展面板右上角 `···` → **从 VSIX 安装…**
3. 选择 `.vsix` 文件，安装后重载窗口

## 使用方法

1. 点击左侧活动栏 **赚钱离场** 图标（折线图标）
2. 在「自选股」视图中点击 **+**，输入股票名称、代码或拼音首字母（如 `gzmt`）搜索并添加
3. 行情会自动刷新，每行显示：**现价 · 涨跌幅 · 成交额**
4. **左键点击**股票行打开详情页；悬停可查看完整 tooltip（含 MA 均线）
5. 右键股票 → **在状态栏显示**，或命令面板搜索 **配置状态栏行情**
6. 命令面板 (`Ctrl+Shift+P`) 可搜索 **添加股票**、**刷新行情**、**查看详情** 等命令

首次使用自带「默认分组」，可直接添加股票，也可先新建分组再添加。

## 界面说明

```
自选股
├── 默认分组 (2 股, 1 组)
│   ├── 子分组 (1 股)
│   │   └── ∧ 贵州茅台    1270.27  -0.61%  16.86亿  长期持有
│   └── ∧∧ 腾讯控股       380.20  +1.02%  65.45亿
└── 科技股 (0 股)
```

| 区域 | 含义 |
|------|------|
| 左侧图标 | 涨跌幅趋势：`∧` / `∧∧` / `∧∧∧`（涨），`∨` 系列（跌），`·`（平） |
| 股票名称 | 证券简称；**左键点击**打开详情页 |
| 右侧数字 | 现价、涨跌幅、成交额；备注与价格提醒（`🔔`）在最末尾 |
| 悬停 tooltip | 现价、成交额、昨收、今开/最高/最低（含相对昨收涨跌幅）、MA5/10/20 等 |

悬停 tooltip 示例：

```
贵州茅台 (sh600519)
市场: 沪
现价: 1270.27 (-0.61%)
成交额: 16.86亿
昨收: 1278.08
今开: 1275.00 (-0.24%)
最高: 1282.88 (+0.38%)
最低: 1268.50 (-0.75%)
MA5: 1275.32
MA10: 1268.45
MA20: 1255.80
```

其中今开、最高、最低后的百分比均为**相对昨收**的涨跌幅。MA 均线见下方说明。

### MA 均线（MA5 / MA10 / MA20）

悬停自选股时，对 **A 股、港股、美股** 按需拉取最近日 K 并计算 MA5 / MA10 / MA20，显示在 tooltip 中。

| 特性 | 说明 |
|------|------|
| 触发方式 | 仅**悬停**时拉取，不做后台轮询，避免与现价刷新抢网络 |
| 缓存 | 当日有效，写入本地缓存；同一交易日重复悬停不再请求 |
| 适用范围 | 支持 K 线的品种（A 股 / 港股 / 美股）；指数、期货、贵金属等无 MA |
| 关闭 | 设置 `take-home.tooltip.maEnabled` 设为 `false` |

首次悬停时 MA 可能尚未加载，稍等片刻或再次悬停即可。

趋势图标规则：

| 涨跌幅 | 图标 |
|--------|------|
| > 5% | 三个 `∧` 或 `∨` |
| > 2% | 两个 |
| > 0% | 一个 |
| = 0% | `·` |

## 添加股票

- 点击标题栏 **+**，或右键分组 → **添加股票**
- 输入框支持**实时搜索**，匹配方式如下：

| 输入类型 | 示例 | 说明 |
|----------|------|------|
| 中文名称 | `茅台`、`腾讯` | 按证券简称搜索 |
| 股票代码 | `600519`、`hk00700` | 支持带市场前缀 |
| 拼音首字母 | `gzmt`、`txkg`、`zgpa` | 名称拼音缩写，如贵州茅台 → `gzmt` |
| 环球指数/期货 | `NKY`、`布伦特`、`A50`、`黄金` | 见下方「环球品种」 |

- 支持 **A 股、港股、美股**，以及**环球指数、国际期货、贵金属**（数据来源：新浪财经）

搜索示例：

```
茅台        → 贵州茅台
600519      → 贵州茅台
gzmt        → 贵州茅台
NKY         → 日经225指数
布伦特      → 布伦特原油
A50         → 富时中国A50期货
黄金        → 纽约黄金 / 伦敦金 / 沪金主连
hf_XAU      → 伦敦金（现货，新浪代码）
```

### 环球品种

以下品种通过**新浪财经**公开接口获取，搜索名称或关键词即可添加：

| 类别 | 搜索示例 | 新浪代码 |
|------|----------|----------|
| 环球指数 | `NKY`、`KOSPI`、`恒生`、`道指` | `b_NKY`、`b_KOSPI`、`b_HSI`、`int_dji` 等 |
| 期货 | `A50`、`布伦特`、`美原油` | `hf_CHA50CFD`、`hf_OIL`、`hf_CL` |
| 贵金属 | `伦敦金`、`纽约黄金`、`沪金` | `hf_XAU`、`hf_GC`、`nf_AU0` |
| 贵金属 | `伦敦银`、`纽约白银`、`沪银` | `hf_XAG`、`hf_SI`、`nf_AG0` |

也可直接输入新浪代码（不区分大小写，如 `b_nky`、`hf_xau`）。

> 国际指数通常有约 15 分钟延迟；期货/贵金属在非交易时段部分字段可能为空；成交额对非 A 股品种意义有限，可能显示 `--`。

## 分组管理

### 新建分组

| 操作 | 效果 |
|------|------|
| 标题栏 **新建分组** | 创建**顶级**分组 |
| 右键分组 → **新建子分组** | 在该分组下创建子文件夹 |

### 重命名 / 删除

右键分组 → **重命名分组** 或 **删除分组**。

删除分组会**同时删除**其下所有子分组和股票，操作不可撤销，请谨慎确认。

### 嵌套示例

```
A股
├── 白酒
│   └── 贵州茅台
└── 科技
    └── 腾讯控股
```

## 拖拽操作

### 股票

| 场景 | 操作 |
|------|------|
| 移到其他分组 | 拖到目标分组或该组内任意股票上 |
| 同组内调顺序 | 拖到同组另一只股票上（**非排序模式**下） |
| 同组内移到末尾 | 拖到同组文件夹名称上（**非排序模式**下） |

### 分组

| 场景 | 操作 |
|------|------|
| 同级调顺序 | 拖到**同级**另一个文件夹上 |
| 移入子级 | 拖到**其他层级**的文件夹上 |
| 提到最外层 | 拖到列表**空白处**，或拖到「默认分组」等**顶级**文件夹上 |

### 排序模式下的限制

开启涨跌幅排序后：

- **不能**在同组内拖拽调整顺序（会提示）
- **可以**拖到其他分组

如需手动排序，请先点击 **排序** → **取消排序**。

## 排序

点击标题栏 **排序**：

- **涨跌幅降序**：涨幅高的在前
- **涨跌幅升序**：跌幅大的在前
- **取消排序**：恢复手动排列顺序

## 备注

右键股票 → **编辑备注**，可为每只股票添加文字说明。

备注显示在每行**最右侧**，例如：`1270.27  -0.61%  16.86亿  长期持有`

留空并确认可清除备注。

## 股票详情页

左键点击自选股，或在右键菜单选择 **查看详情**，在编辑器区域打开详情页。再次点击其他股票会在同一面板切换。

### 行情概览

详情页顶部实时显示现价、涨跌幅，以及涨跌额、成交额、昨收、今开、最高、最低；若已设置备注或价格提醒，也会一并展示。现价随自选股行情刷新间隔自动更新。

### 行情图

| 图表 | 适用品种 | 说明 |
|------|----------|------|
| **分时** | A 股、港股 | 现价 / 均价 / 昨收参考线；A 股含 9:15 起盘前竞价（橙色虚线）；Y 轴显示相对昨收涨跌幅；页面可见时每 60 秒刷新 |
| **日 K** | A 股、港股、美股 | 蜡烛图 + MA5 / MA10 / MA20 均线；鼠标悬停显示 OHLC 与均线值 |
| **周 K** | A 股、港股、美股 | 同日 K，周期为周线 |

鼠标在图表上移动可查看十字光标与数据 tooltip；分时图显示时间与价量，K 线图显示开高低收及日/周 5/10/20 均线。

### 浏览器跳转

对支持的品种，详情页底部提供 **在浏览器中打开完整页面**（东方财富 / 新浪等官方行情页）。右键菜单 **在浏览器中打开** 效果相同。

## 价格提醒

右键股票 → **设置价格提醒**，可分别设置**上限**与**下限**（留空即清除）：

- 现价**涨过**上限或**跌破**下限时，弹出 VS Code 通知
- 已设提醒的股票行末显示 `🔔` 图标；tooltip 与详情页也会展示阈值
- 提醒随行情刷新检测，不额外发起网络请求

上限应高于下限；设置后若当前价已在阈值一侧，需等价格回到另一侧再穿越时才会再次提醒。

## 状态栏行情

底部状态栏可实时显示最多 5 只股票的现价与涨跌幅。

| 操作 | 说明 |
|------|------|
| 点击状态栏行情 | 打开配置面板 |
| 右键股票 → **在状态栏显示** | 将该股票加入状态栏列表 |
| 命令 **配置状态栏行情** | 搜索添加、从自选股选择、清空、开关显示、切换颜色模式 |

颜色模式：

- **彩色**：涨红跌绿（可在设置中自定义颜色）
- **黑白**：跟随 VS Code 状态栏主题色

## 市场概览

状态栏左侧（独立于自选行情）显示 A 股市场统计，**默认开启**。

显示内容：`涨 N  跌 N  平 N  |  成交 X.XX万亿`

| 操作 | 说明 |
|------|------|
| 点击市场概览 | 打开配置面板（开关显示、立即刷新） |
| 命令 **配置市场概览** | 开启/关闭市场概览 |
| 设置 `take-home.marketOverview.enabled` | 控制是否显示 |

悬停 tooltip 还会显示上证指数、深证成指的点位、涨跌幅及沪/深两市各自成交额。

数据来源为东方财富公开接口，仅覆盖 A 股市场。

## 配置

在设置中搜索 `take-home`：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `take-home.refreshInterval` | `5000` | 行情刷新间隔（毫秒），最小 3000 |
| `take-home.riseColor` | `#f14c4c` | 上涨颜色 |
| `take-home.fallColor` | `#73c991` | 下跌颜色 |
| `take-home.statusBar.enabled` | `false` | 是否在状态栏显示行情 |
| `take-home.statusBar.codes` | `["sh000001"]` | 状态栏股票代码列表（需带市场前缀） |
| `take-home.statusBar.colorMode` | `color` | 状态栏颜色模式：`color` 红绿 / `monochrome` 黑白 |
| `take-home.marketOverview.enabled` | `true` | 是否在状态栏显示 A 股市场概览 |
| `take-home.tooltip.maEnabled` | `true` | 悬停 tooltip 是否显示 MA5/MA10/MA20（按需拉取，不后台轮询） |
| `take-home.offHoursRefresh.enabled` | `true` | 全部市场休市时是否降低刷新频率 |
| `take-home.offHoursRefresh.interval` | `60000` | 休市时的刷新间隔（毫秒） |

## 数据来源

行情与搜索数据来自以下公开接口（见 `src/api/stockApi.ts`），仅供个人查看与学习参考：

| 用途 | 接口 |
|------|------|
| 股票搜索 | [腾讯财经 Smartbox](https://proxy.finance.qq.com/ifzqgtimg/appstock/smartbox/search/get) |
| A 股 / 美股行情 | [新浪财经 hq](https://hq.sinajs.cn/list=) |
| 港股行情 | [腾讯财经 qt](https://qt.gtimg.cn/q=) |
| A 股市场概览 | [东方财富涨跌分布](https://push2ex.eastmoney.com/getTopicZDFenBu) · [指数成交额](https://push2.eastmoney.com/api/qt/ulist.np/get) |
| MA 均线 / 日周 K 线 | [东方财富 K 线](https://push2his.eastmoney.com/api/qt/stock/kline/get) · [腾讯财经 K 线](https://web.ifzq.gtimg.cn/appstock/app/fqkline/get) |
| 分时图 | [东方财富分时](https://push2.eastmoney.com/api/qt/stock/trends2/get) · [腾讯财经分时](https://ifzq.gtimg.cn/appstock/app/minute/query) |

- 不保证数据的实时性、准确性与完整性，可能存在延迟或误差
- MA 与 K 线基于日 K 本地计算，仅 A 股 / 港股 / 美股支持；详情页图表按需加载，不影响列表刷新
- 非交易时段部分字段可能为空
- 美股成交额暂不可用（新浪接口未返回该字段）

## 常见问题

**添加后没有价格？**

- 检查网络能否访问上述数据源
- 点击 **刷新** 重试
- 非交易时段部分字段可能为空

**命令找不到？**

- 确认扩展已启用并重载窗口
- 若从 VSIX 安装，请使用最新版本

**数据存在哪？**

- 自选股、分组、备注、价格提醒保存在本机 VS Code 全局存储中，卸载扩展后可能丢失
- MA 均线当日缓存同样保存在本地，跨日自动失效
- 建议定期使用 **导出配置** 备份

**悬停看不到 MA？**

- 确认 `take-home.tooltip.maEnabled` 已开启
- 仅 A 股 / 港股 / 美股支持；指数、期货等无 MA
- 首次悬停需等待拉取，可再次悬停查看

## 与 LeekFund / hgleek 的区别

本插件有意精简，**不包含**：基金、净值走势、持仓盈亏、资讯推送、完整交易终端等。内置详情页提供轻量分时与 K 线（含 MA），市场概览仅提供 A 股涨跌家数与成交额摘要。
