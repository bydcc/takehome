# 赚钱离场

VS Code / Cursor 股票行情看盘插件。支持 A 股 / 港股 / 美股实时行情，自选股分组管理、搜索订阅、拖拽排序、备注与涨跌幅排序，可选状态栏行情展示。

灵感来自 LeekFund（韭菜盒子），专注**精简自选股**，不含基金、K 线、持仓盈亏等重型功能。

## 功能

- **自选股面板** — 侧边栏 TreeView，实时显示现价、涨跌幅、成交额；支持分组嵌套、拖拽排序、备注
- **搜索订阅** — 输入名称或代码实时搜索，一键添加 A 股 / 港股 / 美股
- **涨跌幅排序** — 按涨幅降序 / 跌幅升序排列，可随时取消恢复手动顺序
- **状态栏行情** — 底部状态栏实时显示自选指数或股票价格，支持红绿涨跌色或黑白模式
- **配置导入导出** — 标题栏一键导出 / 导入自选股与分组配置

## 安装

### 从扩展市场安装（推荐）

在 VS Code / Cursor 中打开扩展面板，搜索 **赚钱离场** 并安装。

### 从 VSIX 本地安装

1. 获取 `take-home-x.x.x.vsix` 安装包
2. 扩展面板右上角 `···` → **从 VSIX 安装…**
3. 选择 `.vsix` 文件，安装后重载窗口

## 使用方法

1. 点击左侧活动栏 **赚钱离场** 图标（折线图标）
2. 在「自选股」视图中点击 **+**，输入股票名称或代码搜索并添加
3. 行情会自动刷新，每行显示：**现价 · 涨跌幅 · 成交额**
4. 右键股票 → **在状态栏显示**，或命令面板搜索 **配置状态栏行情**
5. 命令面板 (`Ctrl+Shift+P`) 可搜索 **添加股票**、**刷新行情** 等命令

首次使用自带「默认分组」，可直接添加股票，也可先新建分组再添加。

## 界面说明

```
自选股
├── 默认分组 (2 股, 1 组)
│   ├── 子分组 (1 股)
│   │   └── ∧ 贵州茅台    1270.27  -0.61%  16.86亿  长期持有
│   └── ∧∧ 腾讯控股       380.20  +1.02%  65.45亿
└── 科技股 (0 股)
```

| 区域 | 含义 |
|------|------|
| 左侧图标 | 涨跌幅趋势：`∧` / `∧∧` / `∧∧∧`（涨），`∨` 系列（跌），`·`（平） |
| 股票名称 | 证券简称 |
| 右侧数字 | 现价、涨跌幅、成交额（备注在最末尾） |
| 悬停 tooltip | 现价、成交额、今开、最高/最低（含涨跌幅）等 |

趋势图标规则：

| 涨跌幅 | 图标 |
|--------|------|
| > 5% | 三个 `∧` 或 `∨` |
| > 2% | 两个 |
| > 0% | 一个 |
| = 0% | `·` |

## 添加股票

- 点击标题栏 **+**，或右键分组 → **添加股票**
- 输入框支持**实时搜索**（输入名称或代码即可匹配）
- 支持 **A 股、港股、美股**

搜索示例：`茅台`、`600519`、`腾讯`、`hk00700`、`苹果`

## 分组管理

### 新建分组

| 操作 | 效果 |
|------|------|
| 标题栏 **新建分组** | 创建**顶级**分组 |
| 右键分组 → **新建子分组** | 在该分组下创建子文件夹 |

### 重命名 / 删除

右键分组 → **重命名分组** 或 **删除分组**。

删除分组会**同时删除**其下所有子分组和股票，操作不可撤销，请谨慎确认。

### 嵌套示例

```
A股
├── 白酒
│   └── 贵州茅台
└── 科技
    └── 腾讯控股
```

## 拖拽操作

### 股票

| 场景 | 操作 |
|------|------|
| 移到其他分组 | 拖到目标分组或该组内任意股票上 |
| 同组内调顺序 | 拖到同组另一只股票上（**非排序模式**下） |
| 同组内移到末尾 | 拖到同组文件夹名称上（**非排序模式**下） |

### 分组

| 场景 | 操作 |
|------|------|
| 同级调顺序 | 拖到**同级**另一个文件夹上 |
| 移入子级 | 拖到**其他层级**的文件夹上 |
| 提到最外层 | 拖到列表**空白处**，或拖到「默认分组」等**顶级**文件夹上 |

### 排序模式下的限制

开启涨跌幅排序后：

- **不能**在同组内拖拽调整顺序（会提示）
- **可以**拖到其他分组

如需手动排序，请先点击 **排序** → **取消排序**。

## 排序

点击标题栏 **排序**：

- **涨跌幅降序**：涨幅高的在前
- **涨跌幅升序**：跌幅大的在前
- **取消排序**：恢复手动排列顺序

## 备注

右键股票 → **编辑备注**，可为每只股票添加文字说明。

备注显示在每行**最右侧**，例如：`1270.27  -0.61%  16.86亿  长期持有`

留空并确认可清除备注。

## 状态栏行情

底部状态栏可实时显示最多 5 只股票的现价与涨跌幅。

| 操作 | 说明 |
|------|------|
| 点击状态栏行情 | 打开配置面板 |
| 右键股票 → **在状态栏显示** | 将该股票加入状态栏列表 |
| 命令 **配置状态栏行情** | 搜索添加、从自选股选择、清空、开关显示、切换颜色模式 |

颜色模式：

- **彩色**：涨红跌绿（可在设置中自定义颜色）
- **黑白**：跟随 VS Code 状态栏主题色

## 标题栏按钮

| 按钮 | 功能 |
|------|------|
| 刷新 | 立即拉取最新行情 |
| + | 添加股票 |
| 新建分组 | 创建顶级分组 |
| 排序 | 涨跌幅排序 / 取消排序 |
| 全部折叠或展开 | 一键折叠或展开所有分组 |
| 导出 / 导入 | 备份或恢复自选股与分组配置 |

## 右键菜单

**股票：** 移除 · 移动到分组 · 编辑备注 · 在状态栏显示

**分组：** 添加股票 · 新建子分组 · 重命名 · 删除

## 配置

在设置中搜索 `take-home`：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `take-home.refreshInterval` | `5000` | 行情刷新间隔（毫秒），最小 3000 |
| `take-home.riseColor` | `#f14c4c` | 上涨颜色 |
| `take-home.fallColor` | `#73c991` | 下跌颜色 |
| `take-home.statusBar.enabled` | `false` | 是否在状态栏显示行情 |
| `take-home.statusBar.codes` | `["sh000001"]` | 状态栏股票代码列表（需带市场前缀） |
| `take-home.statusBar.colorMode` | `color` | 状态栏颜色模式：`color` 红绿 / `monochrome` 黑白 |

## 数据来源

行情与搜索数据来自以下公开接口（见 `src/api/stockApi.ts`），仅供个人查看与学习参考：

| 用途 | 接口 |
|------|------|
| 股票搜索 | [腾讯财经 Smartbox](https://proxy.finance.qq.com/ifzqgtimg/appstock/smartbox/search/get) |
| A 股 / 美股行情 | [新浪财经 hq](https://hq.sinajs.cn/list=) |
| 港股行情 | [腾讯财经 qt](https://qt.gtimg.cn/q=) |

- 不保证数据的实时性、准确性与完整性，可能存在延迟或误差
- 非交易时段部分字段可能为空
- 美股成交额暂不可用（新浪接口未返回该字段）

## 常见问题

**添加后没有价格？**

- 检查网络能否访问上述数据源
- 点击 **刷新** 重试
- 非交易时段部分字段可能为空

**命令找不到？**

- 确认扩展已启用并重载窗口
- 若从 VSIX 安装，请使用最新版本

**数据存在哪？**

- 自选股、分组、备注保存在本机 VS Code 全局存储中，卸载扩展后可能丢失
- 建议定期使用 **导出配置** 备份
